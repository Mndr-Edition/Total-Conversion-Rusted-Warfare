# Total-Conversion-Rusted-Warfare
​Sengoku-era total conversion mod for Rusted Warfare. Focuses on asymmetrical balance and highly custom faction mechanics. Built a smart AI with micro-control logic (flanking, raiding) and dynamic scaling. Avoided typical mod bloat: optimized parser queue clearing, high-performance optimization, and deep RPG economy systems.
​Note: This repository does not contain a full playable mod; it contains only the core game logic, AI scripts, and unit .ini files.
